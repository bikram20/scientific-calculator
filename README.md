# Scientific Calculator

A modern, feature-rich scientific calculator built with Next.js, React, and Tailwind CSS.

## Features

- **Basic Operations**: Addition, subtraction, multiplication, division
- **Scientific Functions**:
  - Trigonometric: sin, cos, tan, asin, acos, atan
  - Logarithmic: log (base 10), ln (natural log)
  - Power Functions: square, cube, square root, exponential
  - Special: factorial, reciprocal, pi, e
- **Modern UI**: Beautiful gradient background with responsive design
- **Toggle Mode**: Switch between basic and scientific calculator modes

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the calculator.

### Build for Production

```bash
npm run build
npm start
```

## Usage

1. Click the "Scientific" button to toggle between basic and scientific modes
2. Use number buttons to input values
3. Click operators (+, -, ร, รท) to perform calculations
4. Click "=" to get the result
5. Use "C" to clear all, "CE" to clear entry, or "DEL" to delete last digit
6. In scientific mode, click any scientific function button to apply it to the current value
